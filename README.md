## üí° About The Project

This project bridges the gap between learning theory and practicing code. It organizes a student's file system into an attractive UI.

* **Primary Goal:** To provide a guided interface for the "HTML & CSS Mastery" course.
* **Target Audience:** Beginners learning Web Development.
* **Design Language:** Dark Mode, Neon Gradients (Indigo/Cyan), Blur Effects, and clean typography (Outfit Font).

---

## ‚ú® Key Features

### üé® Design & UI
* **Glassmorphism UI:** Modern, semi-transparent cards with backdrop blur.
* **Fully Responsive:** Grid layouts automatically adjust from Mobile (1 column) to Desktop (2+ columns).
* **Immersive Hero Section:** Full-width background with overlay gradients.
* **Timeline Roadmap:** A visual step-by-step guide to the learning path.

### ‚öôÔ∏è Technical Functionality
* **Single Data Source:** All website content (text, links, phone numbers) is stored in a single JavaScript array (`siteData`). You do not need to touch the HTML structure to update content.
* **Dynamic Rendering:** The dashboard automatically generates project cards based on the folders you define in the script.
* **Smart Navigation:** Includes a "Back to Home" button linking to the parent directory (`../index.html`).
* **Direct File Access:** Students can open specific `.html`, `.css`, or `.js` files directly from the dashboard.
